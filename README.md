# 🚢 Titanic Survival Prediction using Machine Learning

A complete Machine Learning classification project developed as part of my AI/ML Internship.

---

## 📌 Project Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning algorithms.

The project demonstrates the complete ML workflow:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Model Training
- Model Evaluation
- Feature Importance
- Model Saving
- Prediction

---

## 📊 Dataset

- Dataset: Titanic Survival Dataset
- Rows: 418
- Features: 12

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib
- Google Colab

---

## 📈 Exploratory Data Analysis

The notebook includes:

- Survival Distribution
- Gender vs Survival
- Passenger Class vs Survival
- Age Distribution
- Fare Distribution
- Correlation Heatmap
- Boxplots
- Pairplots

---

## ⚙️ Feature Engineering

The following new features were created:

- Passenger Title
- Family Size
- IsAlone
- HasCabin

---

## 🤖 Machine Learning Models

The following models were trained and compared:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

The Random Forest Classifier was selected as the final model based on performance.

## 📊 Evaluation Metrics

- Accuracy

- Classification Report
